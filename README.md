# ggj winter 2024 - smiley meal
created for game jam winter 2024. made in Unity. check out our team's submission [here](https://globalgamejam.org/games/2024/smiley-meal-4)

## HOW TO PLAY
WASD to move around\
E to spawn objects\
F to pick up
